# AI Text Summarizer

A simple Python script that summarizes text using Hugging Face Transformers.  
Built to demonstrate practical NLP skills for AI roles.

## Features
- Summarizes long text into concise, readable output
- Adjustable summary length
- Easy to run from the command line

## Installation
Clone the repo and install dependencies:
```bash
git clone https://github.com/Florentin99/AI-Text-Summarizer.git
cd AI-Text-Summarizer
pip install -r requirements.txt
